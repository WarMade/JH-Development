A collection of high-performance, realism-focused resources for FiveM server environments. This repository contains a suite of interconnected scripts designed to enhance illegal economy mechanics, territory management, and environmental interaction.
🛠 Featured Resources
🏙️ Gang & Territory Ecosystem

    jh-gangwars: A comprehensive drug economy system featuring NPC recruitment, inventory management (bricks/baggies), and integrated shop interfaces.

    jh-turf-war: A robust territory ownership script that tracks regional control and influences gang-related interactions across the map.

    jh-streetstalk-dispatch: An intelligent jurisdictional patrol system. It manages AI police response patterns and lookout intelligence, ensuring law enforcement reacts dynamically based on the current territory status.

📦 Interaction & Environment

    jh-props: A system for interactable world objects, allowing players to engage in detailed looting and scavenging mechanics with synced prop entities.

🏎️ Vehicle Mechanics

    jh-manual: A specialized vehicle resource that introduces manual transmission logic and advanced physics to enhance driving fidelity and immersion.

🚀 Key Features

    Optimized Performance: Built with a "0.00ms resmon" mindset, utilizing PolyZone and qb-target to eliminate heavy distance loops.

    Framework Versatility: Scripts are designed for the QBCore framework, with several available as standalone versions.

    Advanced AI Logic: Features complex NPC behavioral patterns, including "corner selling" logic where dealers maintain static positions for improved server stability.

    3D Assets: Includes custom map assets and prop workflows optimized via Blender 4.0 and Sollumz.

📋 Prerequisites

To ensure these scripts function correctly, the following dependencies (or their equivalents) are generally required:

    ox_lib

    PolyZone

    qb-target or ox_target

📥 Installation

    Clone the repository or download the specific resource folder.

    Move the folder(s) into your server's resources directory.

    Add ensure [resource_name] to your server.cfg.

    Configure the config.lua within each resource to match your server's requirements.
